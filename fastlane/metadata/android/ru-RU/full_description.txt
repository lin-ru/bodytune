Простое приложение для Android для ежедневного мониторинга веса на основе выбранного вами целевого веса: оно отображает ваши результаты на цветной графике тренда: зелёный — значит, вы движетесь по правильному плану, красный — не по цели. Вы также можете отслеживать измерения груди, талии, бедра и бицепса (с помощью графиков измерений).
Поддерживает русский и английский языки (выбирается в зависимости от языка системы).
Не требует никаких разрешений для работы.
Позволяете редактировать запись за любой день.
Позволяет экспортировать /импортировать историю измерений в формате CSV (например для переноса между устройствами).
